# Projeto busca de usuários do GitHub

Esse projeto fazer uma busca de usuários pela API do GitHub, mostrando a foto do perfil, a sua bio, seu seguidores e seguintes, os repositórios e eventos como commits.

## Funcionalidades do site:
- Busca por nome de usuário.
- Renderizar informações como Foto de perfil, bio, seguidos e seguintes, repositórios e eventos.

## Ferramentas Utilizadas
- **HTML**: Para a base do projeto.
- **CSS**: Para estilização.
- **JavaScript**: Para buscar as informações na API e renderizar-las na tela.

## Como rodar o projeto:
1. Clone o repositório do mesmo.
2. Coloque para rodar usando o Live Server.

## Desafios, decisões e informações adicionais
Foi um ótimo projeto para testar minhas habilidades adquiridas até aqui, me senti desafiado tendo que buscar informações na API pela primeira vez, descobrir caminhos para essas informações foi bem interessante. Me sinto pronto para novos desafios e projetos mais complexos. É um projeto um tanto quanto simples, porém nos da uma ótima base para como interagir com API's e também ao mesmo tempo estilizar de forma responsiva o projeto. Espero que tenham gostado e até breve.
